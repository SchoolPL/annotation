Небольшой экскурс по проекту:
Child - расширяет класс Parent. Содержит аннотации с одним значением и аннотации маркеры
IPrintF - интерфейс помеченный аннотацией, который говорит что он функциональный. Содержит в себе так же Вариант использования повторяющейся аннотации.
MyAnnotation - самописная аннотация с одним дефолтным значением value
MyMarker - аннотация маркер
Parent - класс родитель. Имплементриует несколько интерфейсов маркеров и помечен аннотацие маркером.
         Содержит аннотации маркеры, аннотации с одним значение и полные аннотации
ProjectBody - класс с аннотацией от lombok
Tests - тестовый класс с одним методом.

В папке таргет при билде создаются файлы проекта.
1. Можете посмотреть какие классы там лежат (там так же создались методы lombok, про которые говорил Лёша.)
2. Так же в этой папке есть allure-result - папка с файлами для построения отчёта (там только 2 строчки, но всё же)
(можете запустить справа во вкладке Maven->Plugins->allure-> allure:serve формирование отчёта)

javadocs лежит в папке src->test->java->javadocs. Для просмотра откройте файл index.html в браузере,
либо сгенерируйте заново сами Tools->Generate JavaDoc. Установите радиоботтон в Whole Project,
укажите output directory (для быстрого поиска пути до файла можете перейти в в любой класс, скажем Child
и нажать горячие кнопки Shift+Ctrl+S (будет скопирован путь до этого файла в буфер обмена), далее просто вставьте в директорию
и добавьте папку javadocs
